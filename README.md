## My-Hobby-Web Project

나의 취미를 소개하는 자기소개 웹 사이트 <br/>
><https://yundol777.github.io/my-hobby-web/>

<br/>

### 💻 프로젝트 소개

>HTML 과 CSS를 공부하며 제작한 자기소개 사이트 입니다 <br/>
>좋아하는 취미들을 직접 그려서 웹사이트를 구성했습니다 <br/>
>그림을 선택하면 각 취미에 대한 디테일한 정보를 볼 수 있습니다
